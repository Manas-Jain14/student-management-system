# Student Management System (C++)

 Description
This is a console-based Student Management System built using C++. It allows users to manage student records efficiently.

# Features
- Add Student
- Display All Students
- Search Student
- Delete Student

##Technologies Used
- C++
- Object-Oriented Programming (OOP)
- STL (vector)
- File Handling (ifstream, ofstream)

## How to Run
open new terminal
g++ main.cpp Student.cpp -o app
.\app.exe
